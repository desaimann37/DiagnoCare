## Installation

1. Clone the Repository

```bash
  git clone https://github.com/ishapaghdal301/DiagnoCare.git
  cd DiagnoCare
```

2. Backend Setup 
```bash
  cd backend
  pip install -r requirements.txt
  flask run
```

3. Frontend Setup
```bash
  cd frontend
  npm install
  npm start
```
